computeFeatures(I1, I2, I3):

Compute the features dependent on the right Cauchy-Green strain invariants. Note that the features only depend on I1 and I3.

Input Arguments

  • I1 - 1st invariant

  • I2 - 2nd invariant

  • I3 - 3rd invariant

Output Arguments

  • x - features

getNumberOfFeatures():

Compute number of features.

Input Arguments

  • none

Output Arguments

  • features.shape[1] - number of features
